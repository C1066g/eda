## 一、计算题（共3题，30分）

**Q1** (10分) 长度10^8bit数据块，带宽1Mbps信道，发送时延？光纤1000km，传播速度2×10^8m/s，传播时延？
- **发送时延** = 10^8 / 10^6 = **100s**
- **传播时延** = 10^6 / (2×10^8) = **5ms**

**Q2** (10分) 点对点链路50km，传播速度2×10^8m/s，带宽应为多少使传播时延和发送100B/512B分组的发送时延一样？
- Tp = 5×10^4 / (2×10^8) = 2.5×10^-4s
- B1 = 800 / (2.5×10^-4) = **3.2Mbps**
- B2 = 4096 / (2.5×10^-4) = **16.384Mbps**

**Q3** (10分) 传输距离1000km，传播速率2×10^8m/s。
- (1) 数据10^7bit，速率100kbps → 发送时延=**100s**，传播时延=**5ms**
- (2) 数据10^3bit，速率1Gbps → 发送时延=**1μs**，传播时延=**5ms**

## 校对说明

- **17 题答案全部正确**；得分 95，约错 1 题
- **Q8 ⚠️ 必考**：速率 k/M/G/T 用 **10³**；数据量 K/M/G/T 用 **2¹⁰**（1KB=1024B，1kbps=1000bps）
- **Q10 端到端**：传输层是第一个提供端到端（进程到进程）服务的层
- **Q11 木桶效应**：实际速率 = min(发送端, 链路, 接收端) = **10Mbps**
- **Q14 判断**：层间只**提供服务**，协议是层内对等实体间的规则
