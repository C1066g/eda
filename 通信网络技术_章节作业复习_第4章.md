## 校对说明

- **28 道单选题答案全部正确**；计算/填空/论述与作业标准一致
- **Q2 易错**：协议字段表示**上层协议**(TCP=6)，不是 IP 版本；版本在版本字段(值为4)
- **Q12 可分配 IP**：192.46.10.0 是 C 类**网络地址**(主机号全0)；127.x 是环回；256>255 非法
- **Q13 分片**：4500B TCP 段 + 20B IP 首 → 每片数据≤1480B(MTU1500) → **4 片**
- **Q32 分片**(3820B/MTU1420)：每片数据≤1400B → **3 片**；片偏移 0/175/350
- **ARP**：仅**发送**时、且缓存无目的 IP 的 MAC 映射，才发 ARP 请求 ⚠️
- **IP 全程不变**(无 NAT)：源/目的 IP 不变，MAC 逐段改变
- **RIP**：收到邻居距离 +1 再比较；距离 **16 = 不可达**；更优路由才更新(N3/N4 保持原路由)
