## 常见端口号

| 协议 | 端口 | 运输层 |
|------|------|--------|
| HTTP | 80 | TCP |
| HTTPS | 443 | TCP |
| FTP控制 | 21 | TCP |
| FTP数据 | 20 | TCP |
| SMTP | 25 | TCP |
| POP3 | 110 | TCP |
| IMAP | 143 | TCP |
| DNS | 53 | UDP(查询)/TCP(区域传送) |
| DHCP | 67/68 | UDP |
| Telnet | 23 | TCP |

## 核心概念

- **DNS查询方式**：递归（一条龙服务）vs 迭代（一步一步来）
- **FTP**：控制连接(21)持久 + 数据连接(20)按需建立
- **SMTP**：发邮件（推），**POP3/IMAP**：收邮件（拉）
- **DHCP DISCOVER**：源0.0.0.0，目的255.255.255.255（广播）
- **HTTP无状态** → Cookie解决会话跟踪

## 校对说明

- **25 题答案全部正确**；得分 92，错题可能在重复题或协议细节
- **Q9 与 Q14**：同一题 DHCP DISCOVER，答案均为 **0.0.0.0 → 255.255.255.255**（选项字母 D/C 因卷面选项顺序不同）
- **Q13 ⚠️**：SMTP 基于 **TCP**（有连接、可靠），不是 UDP
- **Q19 ⚠️**：域名与 IP **不是**一一对应（负载均衡/虚拟主机）
- **Q6**：UDP 是**运输层**，不属于应用层
- **FTP 端口**：服务器控制 **21**、数据 **20**；**客户端**控制连接用**随机端口**（不是 20）⚠️
- **邮件**：发 **SMTP**，收 **POP3/IMAP**；两者都基于 TCP
